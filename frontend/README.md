# Development server

Pastikan anda sudah menginstall node versi `^22.x.x` dan pnpm `^8.x.x`.

## Install seluruh dependencies

```bash
pnpm install

```

## Persiapan .env.local

Jika sudah anda dapat membuat file `.env.local` berdasarkan `.env.example` yang tersedia.

## Running secara lokal

```bash
pnpm dev

```
